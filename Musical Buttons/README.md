# Musical Buttons
MATLAB script designed to respond with light/sound when corresponding green, blue, yellow, and red buttons are pressed.

### Materials:
- Arduino Uno
- Piezo Speaker
- 4 Pushbuttons (Green, Blue, Yellow, Red)
- 4 LEDS (Green, Blue, Yellow, Red)
- 5 resistors
