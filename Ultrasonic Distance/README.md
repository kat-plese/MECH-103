# Ultrasonic Distance
MATLAB script designed to utilize an ultrasonic sensor to detect distances through SONAR.

### Materials:
- Arduino Uno
- HC-SR04 Ultrasonic Sensor 
- Piezo Speaker
- 5 LEDS (2 Red, 1 Yellow, 2 Green)
- 5 resistors
